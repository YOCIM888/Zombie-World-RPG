## 一、阵营总览

游戏分为三大路线，其中**曙光阵地**与**末日城堡**互斥：

| 路线 | 核心NPC | 终极目标 | 对立路线 |
|------|---------|---------|---------|
| 曙光阵地 | V小姐、苏小涵、莉莉丝、沐苗苗、首领 | 曙光队长徽章 / 农神结局 / 爱恨结局 | 末日城堡 |
| 末日城堡 | 国王、皇后 | 储君身份牌 / 爱恨结局 | 曙光阵地 |
| 中立 | 陈博士、露露薇、柳如烟、马三、老狼、老马 | 火箭结局 / 新势力结局 / 人鱼结局 / 当枪结局 / 盘中餐结局 / 私奔结局 | 无 |

**关键规则**：
- 加入曙光阵地（获得曙光徽章）后，无法在城堡银行借贷、打工或办理贵族身份
- 获得城堡身份牌后，阵地NPC会拒绝与你交互
- 但**城堡通行证**不会触发阵营排斥，也不会因持有曙光队长徽章而消失

## 二、曙光阵地线路

### 2.1 V小姐线

**位置**：曙光阵地 → 与V小姐对话

V小姐是阵地最强的战斗教官，任务线以"战斗物资交换武器"为核心，奖励从低级到顶级递进。

| 任务 | 好感要求 | 提交物品 | 奖励 |
|:----:|:-------:|---------|------|
| V1 初见任务 | 0 | 任意1食物+1饮品 | 小匕首 |
| V2 止血训练 | 30 | 止血带×5 | 指虎刀 |
| V3 血清任务 | 60 | 抗感染血清×8 | 龙抄剑 |
| V4 困难试炼 | 90 | 战地医疗箱×8 | P226 + 9×19mm×50 |
| V5 急救物资 | 120 | 医用急救包×6 + 抗感染血清×6 | UZI + 9×19mm×50 |
| V6 终极试炼 | 150 | 所有药品各2个 | M4A1 + 5.56×45mm NATO×50 |

**V6所需药品清单**（各2个）：
创可贴、止血带、抗生素、抗感染血清、手术包、万能针剂、医用急救包、战地医疗箱

**好感提升方式**：每日对话+1、送礼+1、完成任务

**位置**：曙光阵地 → 与莉莉丝对话

| 回收远程武器 | 1件武器 | 获得2根香烟 |
| 维修枪支 | 10根香烟 | 远程武器+50%完整度 |
| 修复弓弩 | 5根香烟 | 弓/弩+50%完整度 |
| 清除感染 | 10根香烟 | 感染值-50% |

**好感150奖励**（一次性）：香烟×20 + 9×19mm×50

### 2.4 沐苗苗线（⭐核心剧情）

**位置**：曙光阵地 → 与沐苗苗对话

这是全游戏**最感人的任务线**，也是通往结局九"我不是农神"的唯一路径。

#### 前置条件链

```
沐苗苗好感150 → 解锁秘密 → 获得农业大师徽章
                                    ↓
                    前往瞭望塔 → 花圈墓碑 → 查看（需徽章）
                                    ↓
                              挖掘（需装备铁铲）
                                    ↓
                            获得苗苗日记本
                                    ↓
                    回到沐苗苗处 → 解锁"埋没的过去"
```

#### 第一步：获取农业大师徽章

**条件**：沐苗苗好感度 ≥ 150

好感提升方式：
- 购买种子（1香烟→3颗种子，好感+少量）
- 照看实验田（每日1次，2回合，好感+1~3，获1颗随机种子）
- 每日对话+1、送礼+1

**奖励**：农业大师徽章

#### 第二步：获取苗苗日记本

**地点**：山顶废弃瞭望塔 → 选择"花圈墓碑"

1. 选择"查看"：需农业大师徽章
2. 选择"挖掘"：需装备铁铲 → 获得**苗苗日记本**

#### 第三步：任务链"埋没的过去"

| 任务 | 需求 | 奖励 |
|:----:|------|------|
| 1 一起去野炊 | 胡萝卜×3 + 土豆×3 | 好感+20 |
| 2 跨时空的思念 | 高度白酒×5 + 黑麦×10 | 好感+30 |
| 3 扩大试验田 | 15种不同作物各1个 | 好感+50，**原野之弓** + 箭矢×30 |

**原野之弓**（52伤/52%暴击/520完整度/箭矢/传奇）——剧情唯一武器。

#### 结局触发

完成任务3后，**立即触发**结局九"我不是农神"。

### 2.5 阵地首领线（⭐核心剧情）

**位置**：曙光阵地 → 与阵地首领对话

这是全游戏**横跨两大阵营**的核心主线，串联了录音笔、首领、皇后三条故事线。

#### 前置条件链

```
曙光队长徽章 → 河边露营地 → 尸臭的帐篷 → 奇怪录音笔
                                            ↓
                                    交给阵地首领
                                            ↓
                                    任务1：给妹妹的信
                                            ↓
                                    交给皇后（迟到的道歉）
                                            ↓
                                    获得皇后的回信
                                            ↓
                                    任务2：喝酒读信
                                            ↓
                                    任务3：酒精中毒
                                            ↓
                                    获得定情信物 → 解锁皇后线
```

#### 第一步：获取奇怪录音笔

**地点**：河边露营地 → 选择"尸臭的帐篷"

- 需持有曙光队长徽章才能进入
- 进入后选择"翻找"，获得**奇怪录音笔**

#### 第二步：首领任务链

| 任务 | 条件 | 需求 | 奖励 |
|:----:|------|------|------|
| 1 关于录音笔 | 持有奇怪录音笔 | — | 给妹妹的信，好感+10 |
| 2 喝酒读信 | 持有皇后的回信 | 高度白酒×5 | 好感+20 |
| 3 酒精中毒 | 完成2 | 万能针剂×3 | 9×19mm×100 + **定情信物**，好感+30 |

---

### 2.6 曙光队长徽章

**条件**：V小姐、苏小涵、莉莉丝、首领好感度全部达到150，且**无任何城堡身份**

**奖励**：曙光队长徽章 + Vector + 9×19mm×60

**隐藏效果**：
- 打工和乞讨产出翻倍
- 莉莉丝所有服务永久半价
- 无法获得任何城堡身份
- 可进入河边露营地"尸臭的帐篷"

## 三、末日城堡线路

### 3.1 国王线

**位置**：末日城堡 → 与国王对话

**前置**：需先购买贵族身份牌（25根香烟）

| 任务 | 需求 | 奖励 | 爵位 |
|:----:|------|------|:----:|
| 1 皇后的烦恼 | 葡萄糖服液×10 | 子爵身份牌 | rank 2 |
| 2 国王的宴会 | 高度白酒×10 | 伯爵身份牌 | rank 3 |
| 3 国王的嗜好 | 香烟×10+百味啤酒×5+动力啤酒×5 | 侯爵身份牌 | rank 4 |
| 4 长生不老 | 万能针剂×3 | 公爵身份牌 | rank 5 |
| 5 清除异己 | 暗杀阵地首领 | 储君身份牌 | rank 6 |

**⚠️ 关键抉择**：任务5要求暗杀阵地首领。首领战力等同丧尸之王（HP 400，伤害 110，闪避 25%）。暗杀成功后：
- 首领死亡，阵地功能受限
- 无法再与V小姐、苏小涵、莉莉丝交互
- 无法获得曙光队长徽章
- 触发结局三"权力的沉溺"

**地下钥匙**：达到储君(rank 6)后，可向国王上缴一把AWM狙击步枪换取地下钥匙。地下区域有小核能发电机，是陈博士火箭链的关键道具。

## 五、中立线路

### 5.1 伙伴系统

#### 柳如烟（半感染的女人）

**地点**：市中心综合商场

**发现文本**：`你发现角落躺着一个半感染的女人，她还有一丝意识。`

**救援条件**：30天内注射抗感染血清×3

**救援后效果**：每天自动搜集物资（食物/饮品/水果/弹药）

## 十、推荐路线规划

### 路线A：全剧情体验（推荐首次游玩）

├─ 莉莉丝好感提升（利用回收/维修服务）
